# JSON Space Analyzer

## Usage

Go to https://json-space-analyzer.com and drop in a JSON file!

## Development

To install dependencies:

```bash
bun install
```

To run in development mode:

```bash
bun run dev
```

This will run a small web server and `bun build` in watch mode.
